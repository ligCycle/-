# แบบสอบถามความพึงพอใจ + ระบบหลังบ้าน

ระบบเก็บคำตอบแบบสอบถามไว้ส่วนกลาง พร้อมหน้า Admin สำหรับดูรายชื่อผู้กรอก
สร้างด้วย Node.js ล้วน — **ไม่ต้องติดตั้งแพ็กเกจใด ๆ เพิ่ม**

## วิธีเริ่มใช้งาน

1. เปิด PowerShell หรือ Command Prompt ไปที่โฟลเดอร์นี้
2. รันคำสั่ง:

   ```powershell
   node server.js
   ```

3. จะเห็นข้อความว่าเซิร์ฟเวอร์พร้อมใช้งาน แล้วเปิดเบราว์เซอร์:

   - **หน้ากรอกแบบสอบถาม** → http://localhost:3000/
   - **หน้า Admin (ดูผู้กรอก)** → http://localhost:3000/admin

4. หยุดเซิร์ฟเวอร์ด้วยการกด `Ctrl + C`

## การเข้าสู่ระบบ Admin

เมื่อเปิด `/admin` เบราว์เซอร์จะถามชื่อผู้ใช้และรหัสผ่าน (ค่าเริ่มต้น):

- **ผู้ใช้:** `admin`
- **รหัสผ่าน:** `changeme123`

> ⚠️ **สำคัญ:** เปลี่ยนรหัสผ่านก่อนใช้งานจริง โดยแก้ที่ไฟล์ `server.js` บรรทัด `ADMIN_PASS`
> หรือกำหนดผ่าน environment variable ตอนรัน เช่น:
>
> ```powershell
> $env:ADMIN_USER="myname"; $env:ADMIN_PASS="รหัสลับของฉัน"; node server.js
> ```

## ความสามารถของหน้า Admin

- ดูรายชื่อผู้กรอกทั้งหมด: ชื่อ-นามสกุล, อีเมล, เบอร์โทร, คะแนน, ข้อเสนอแนะ, วันเวลาที่ส่ง
- สรุปสถิติ: จำนวนคำตอบ, คะแนนเฉลี่ย, สัดส่วนความพึงพอใจ (4-5 ดาว)
- ค้นหาผู้กรอกด้วยชื่อ / อีเมล / เบอร์
- คลิกอีเมลเพื่อส่งเมล / คลิกเบอร์เพื่อโทร
- ดาวน์โหลดคำตอบทั้งหมดเป็นไฟล์ CSV (เปิดใน Excel ได้ รองรับภาษาไทย)

## ข้อมูลถูกเก็บที่ไหน

คำตอบทั้งหมดถูกเก็บในไฟล์ `data/responses.json`
สำรองข้อมูลได้ง่าย ๆ ด้วยการคัดลอกไฟล์นี้เก็บไว้

## ให้คนอื่นในวงเน็ตเดียวกันกรอกได้

- หาไอพีเครื่องคุณด้วยคำสั่ง `ipconfig` (ดู IPv4 Address เช่น `192.168.1.20`)
- ให้ผู้อื่นเปิด `http://192.168.1.20:3000/` (ต้องอยู่ในเครือข่าย Wi-Fi/LAN เดียวกัน)
- หากไฟร์วอลล์ Windows ถาม ให้อนุญาต (Allow access)

## โครงสร้างไฟล์

```
survey-backend/
├── server.js          เซิร์ฟเวอร์ + API (แก้รหัสผ่าน/พอร์ตที่นี่)
├── package.json
├── public/
│   ├── index.html     หน้ากรอกแบบสอบถาม
│   └── admin.html     หน้า Admin
└── data/
    └── responses.json ที่เก็บคำตอบ (สร้างอัตโนมัติ)
```
